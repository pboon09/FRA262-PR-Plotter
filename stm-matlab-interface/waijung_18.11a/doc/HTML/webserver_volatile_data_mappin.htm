(Obsolete block)

User Interface

How this block appears in a Simulink model?

What can be configured?

Configuration item

Selectable option/ Value

Description

Volatile Data Mapping

(Select volatile data name to remap)

Select volatile data storage, to available for query on web browser.

Access mode

ReadWrite/ReadOnly

ReadWrite - The selected volatile data storage can change value from url.

ReadOnly - The selected volatile data storage cannot change by url, it only available form read.

Enable callback on CGI varet

Checked | Unchecked

Checked - callback function when value changed by url.

Unchecked - disable callback.

Enable 'name=value' pair on CGI varset.

Checked | Unchecked

The request value return to url will include name.
