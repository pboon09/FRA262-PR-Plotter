Minimum Requirements

Software

Absolute Minimum

1.Microsoft Windows XP SP3 or later. (Windows 7 recommended.)
2.Mathworks Inc. software suite consisting of the following toolboxes and blocksets.

Release R2009a - R20010b

Release R2011a or later

Matlab

Simulink

Real-Time Workshop

Real-Time Workshop Embedded Coder

Matlab

Simulink

Simulink Coder

Embedded Coder

Note: Waijung Blockset supports both 32-bits and 64-bits operating systems, as well as, Matlab 32-bits and 64-bits versions.

3.Either one of the following C Code Compilers.
a)GNU Tools for ARM Embedded Processors
[There is no need to download the compiler. It is already available in Waijung package.]
b)Keil Microcontroller Development Kit for ARM (MDK-ARM) version 4.60 or later.
[Free MDK-Lite Edition (32KB Code Size Limitation) is available for download. It does not require a serial number or license key.]
c)IAR Embedded Workbench for ARM (EWARM) version 6.4 or later
[Free KickStart edition of IAR Embedded Workbench for ARM (32KB Code Size Limitation) is available for download. Registration required.]
See procedures to manual compile and build with EWARM.
4.Free STMicroelectronics STLink Utility to enable automatic flash programming directly after the build process as well as in-circuit debugging.
[Direct download]
(See how to (http://www.trickday.com/2012/09/how-to-install-unsigned-drivers-in-windows-8/) install STLink Utility driver with Windows 8)

Hardware

Waijung Blockset and STM32F4 Target support a complete range of STM32F4 family of Hi-Performance & DSP MCU.
However, the cheapest and fastest way to get started is to test out the blockset with the STM32F4DISCOVERY Kit which is a low-cost development Kit from STMicroelectronics.

 

For as low as 15 US dollars, STM32F4DISCOVERY Kit can be purchased directly from Aimagin.
