Features

1.Design specifically for and compatible with FiO 2 and STM32F4DISCOVERY
2.Enable various add-on expansion boards such as:
a.aMG SQLite Database Server for embedded database
b.aMG USB Converter - N for real-time Hardware in the Loop Simulation (upto 2kHz)
c.aMG Ethernet INF for LAN interface
d.aMG CANINF for CAN BUS Interface
e.aMG 232/485 INF for RS232 / RS485 Interface
f.aMG High Precision RTC for high precision real-time clock
3.Onboard 3.3V and 5V Power Supply Regulator
4.Accept 9 - 25 VDC Input Voltage
5.4 User LEDs
6.3 User Push Buttons
7.8 User Dip Switches
8.I2C and SPI ports
9.Fit in an industrial grade enclosure
10.Matlab / Simulink support with Waijung Blockset
11.RoHS
