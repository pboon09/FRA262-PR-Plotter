aMG SQLite Database Server

Features

1.SQLite 3  Database Server
2.ARM Cortex-M4 FPU processor (STM32F4), 168MHz
3.Two operating modes (UART-SQLite database and USB mass storage mode).
4.Database mode (enable database query from any microcontrollers via UART port)
a)USB Mass Storage Device (Flash Drive) mode (enable database query from any PC via USB port)
b)Micro SD Card slot
5.Matlab / Simulink support with Waijung Blockset
6.Basic IO Protection against spikes.
7.Operating Voltage: 3.3V
8.Firmware Upgradable
9. RoHS

Applications

1. Low-cost stand-alone database server

2. Network based monitoring and control

3. Data logger
