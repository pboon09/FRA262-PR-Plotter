aMG High Precision RTC










Features

1.Based on Maxim DS3231 Extremely Accurate RTC Chip
•Accuracy ±2ppm from 0°C to +40°C
•Accuracy ±3.5ppm from -40°C to +85°C
•Battery Backup Input for Continuous Timekeeping
•Industrial Operating Temperature Ranges: -40°C to +85°C (DS3231SN)
•Low-Power Consumption
•Real-Time Clock Counts Seconds, Minutes,Hours, Day, Date, Month, and Year with Leap YearCompensation Valid Up to 2100
•Two Time-of-Day Alarms
•Programmable Square-Wave Output
•Fast (400kHz) I2C Interface
•3.3V Operation
•Digital Temp Sensor Output: ±3°C Accuracy
•Register for Aging Trim
•RST Output/Pushbutton Reset Debounce Input
•Underwriters Laboratories (UL) Recognized
2.aMG High Precision RTC module ships with button battery holder and right angle pin header. However, these are not soldered just in case user need different configurations.
3.Battery is not included in the package.

4. ROHS Compliant

Applications

1.Extremely accurate real - time clock for remote monitoring system under adverse condition
2.Generic Real-Time Clock module
