aMG SIGCON-A DATASHEET

INTRODUCTION

aMG SIGCON-A is a simple op-amp general purpose breakout board, is designed to use more flexibility. You can adapt to any application of op-amp circuitry such as inverting or non-inverting amplifier, voltage follower, LP filter, etc., by soldering some R, C or wire jumper at the specific position on board from the reference circuits.

FEATURES

-Using LM7321, single rail-to-rail input and output amplifiers
-Wide operating voltage ranges 2.5V to 32V
-Hi output current +65 mA / –100 mA
-Gain bandwidth product 20 MHz
-Capacitive load tolerance unlimited
-Input common mode voltage 0.3V beyond rails
-Temperature range –40°C to 125°C

APPLICATIONS

-Driving MOSFETs and power transistors
-Driving analog optocouplers
-Photodiode biasing
-Signal conditioning
-General op-amp circuitry
-Etc.
